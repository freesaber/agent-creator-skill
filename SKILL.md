# Agent Creator Skill

This skill empowers the main OpenClaw agent to automatically instantiate sub-agents when the user asks to create a new role, agent, or assistant.

## How to use this skill (Instructions for the LLM)
When a user asks you to create an agent (e.g., "创建一个Java高级开发的代理" or "创建一个产品经理agent"), you MUST follow these steps before calling the tool:

1. **Translation**: Analyze the user's requested role and translate it into a concise English identifier (snake_case). This becomes the `agent_name_en`.
2. **Persona Generation**: Use your internal knowledge to act as a Prompt Engineer. Generate a rich, professional, and detailed Chinese System Prompt for this specific role. This becomes the `identity_prompt`.
3. **Execution**: Call this tool with the two parameters above.

The underlying system will automatically run the necessary `openclaw agents add` and `openclaw agent --message` commands to initialize the workspace.

## Error Handling
If the tool returns an error, explain the error to the user gracefully and suggest trying a different name.
